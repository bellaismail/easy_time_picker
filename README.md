**easy_time_picker** - Cupertino-style time picker for Flutter with smooth wheel scrolling and native iOS design.

## Features

- Native iOS-style wheel picker with smooth 3D animations
- Full theming for colors, fonts, and divider styling
- Lightweight with zero dependencies
- Accessibility and haptic feedback support

## Getting started

Add `easy_time_picker` to your `pubspec.yaml` dependencies and run `flutter pub get`.
